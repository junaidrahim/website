I try my best to not write the final draft published here with AI.

The temptation is very hard to avoid, but I want to grow my ability to articulate complex ideas in ways I see them.

I am still very divided on how to use AI to write more in a way that does not make my readers feel cheated, but I do want my blogs to read like a [Ron Swanson](https://parksandrecreation.fandom.com/wiki/Ron_Swanson) chair.

I use AI heavily to do research, build prototypes or diagrams used in a post. I use claude code to make the code changes that go into maintaining this site.
